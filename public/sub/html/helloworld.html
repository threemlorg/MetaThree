Hello World!

The code below shows a basic ThreeML implementation. As can be seen the code contains both standard Html tags as those specific to ThreeML.

Every 3D object are specified within the <three> tags:

    <html>
       <body>
           <three>
               <spheregeometry name="globe" position="0 0 -2" scale="0.1" widthSegments="30" heightSegments="30">
                   <meshphongmaterial id="blueish" color="1 1 1" url="./images/earth-topo.png" rotation="0 0 20"></meshphongmaterial>
                   <rotate axis="0 0.002 0"></rotate>
               </spheregeometry>
               <TextGeometry position="0 0.3 -2" text="Hello">
                   <meshphongmaterial id="black" color="0 0 0"></meshphongmaterial>
               </TextGeometry>
               <directionallight target="globe" intensity="1.6" position="-2 0 0"></directionallight>
           </three>
           <script>
               var threeml;
           </script>
           <script type="module">
               import { ThreeML } from './threeml/threeml.js';
               threeml = new ThreeML();
               threeml.parseThree();
           </script>
    <body>
   </html>

A General JavaScript part defines the global 'threeml' variable. Although not strictly necessary for this page this is often handy for a little more advanced implementation. In a separate JavaScript part with type='module' the threeml module is loaded, and the variable is set to an instance of threeml. Then the parseThree() function is called, which will look for <three> tags in the DOM, and start interpreting the inner tags.
